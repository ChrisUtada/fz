## 二、P0（必须修）

### 1. `placement_manager.gd:63-66` — remove_placed 复活 bug（已验证 · ✅已提交推送 a862615）
`queue_free()` 不会立即把节点移出树，紧接着的 `save_placements()` 遍历 `get_children()` 时被删节点仍在，仍被写入存档 → 下次 `restore()` 复活（摆放物删不掉，越删越多）。

**修法 A（推荐）**：save 时跳过待删节点
```gdscript
for c in _container.get_children():
    if c is Placeable and c.data != null and not c.is_queued_for_deletion():
        # 写入存档
```

**修法 B**：先移出树再释放
```gdscript
remove_child(p)
p.queue_free()
save_placements()
```
